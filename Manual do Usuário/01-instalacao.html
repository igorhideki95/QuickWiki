01 - Instalacao e Preparacao

Instale o QuickWiki a partir do source e confirme a CLI canonica antes de seguir para o primeiro crawl.

Comandos

python -m pip install .
quickwiki --help
quickwiki --validate-site-profiles

Se o Windows ainda nao reconhecer quickwiki, abra um novo terminal. Enquanto ajusta o PATH do Scripts do Python do usuario, use python -m quickwiki --help ou python run_scraper.py --help.

O que confirmar

  • O comando `quickwiki` esta disponivel.
  • O fallback `python -m quickwiki` tambem funciona.
  • Os perfis built-in oficiais validam sem erro.
  • `python run_scraper.py --help` segue funcionando como compatibilidade.
