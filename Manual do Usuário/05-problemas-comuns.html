05 - Problemas Comuns

Use esta pagina quando algo nao sair como esperado durante a instalacao, validacao ou crawl.

Comandos uteis

quickwiki --validate-site-profiles
quickwiki --serve-only --output-dir output
quickwiki --fresh

Checagens rapidas

  • Confirme se voce instalou com `python -m pip install .`.
  • Se `quickwiki` nao for reconhecido, abra um novo terminal ou use `python -m quickwiki` como fallback imediato.
  • Revise `output/logs/scraper.log` quando o crawl parar cedo.
  • Use `quickwiki --list-site-profiles` se o perfil nao estiver aparecendo.
  • Se faltarem perfis, revise `--profiles-dir` ou ajuste `QUICKWIKI_ROOT` para a clone correta.

Volte ao indice principal quando quiser repetir o fluxo.
