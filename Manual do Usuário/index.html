Antes de Rodar

Checklist rapido

  • Voce esta dentro da pasta do projeto.
  • O source foi instalado com `python -m pip install .`.
  • `quickwiki --help` funciona, ou `python -m quickwiki --help` se o PATH ainda nao estiver pronto.
  • `quickwiki --validate-site-profiles` retorna sucesso.
  • `quickwiki --gui` abre a interface local.
Dica Pratica

Primeiro caminho ideal

O melhor comeco agora e abrir a GUI local depois da validacao de perfis. Ela reduz a friccao operacional, deixa os parametros mais visiveis e facilita a navegacao entre logs, resumo e espelho gerado.
